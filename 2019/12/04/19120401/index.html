android中UI控件整理

发表于 2019-12-04 更新于 2021-04-16 分类于 android学习

Android学习笔记

一、布局管理器(ViewGroup继承来)

1、相对布局RelativeLayout
    RelativeLayout上设置属性
        android:gravity="center/..."
        android:ignoreGravity="id"
    子组件设置属性
        android:layout_above            android:layout_alignParentBottom
        android:layout_below            android:layout_alignParentLeft
        android:layout_toLeftOf            android:layout_alignParentTop
        android:layout_toRightOf        android:layout_alignParentRight

2、线性布局LinearLayout    
    LinearLayout上设置属性
        android:orientation="vertical/horizontal"
        android:gravity="center/..." 多个属性通过|分隔，如right|bottom居右下角
    子组件设置属性
        android:layout_width="1/..." 组件分配剩下的空间

3、帧布局FrameLayout
    FrameLayout上设置属性
        android:foreground="..." 设置顶层图像
        android:foregroundGravity="..." 设置顶层图像位置

4、表格布局TableLayout
    通过<TableRow></TableRow>添加表格行

5、网格布局GridLayout
    GridLayout设置属性
        android:columnCount="5/..."    设置最大列数
        android:rowCount="5/..."    设置最大行数
        android:orientation="vertical/..."    设置默认排列模式
    子组件的分布
        android:layout_column="2/..."    设置子组件位于网格的第几列
        android:layout_columnSpan="2/..." 设置子组件跨几列
        android:layout_columnWeight        设置子组件分配剩余空间的权重
        android:layout_gravity            设置子组件怎么排列的
        android:layout_row="2/..."    设置子组件位于网格的第几行
        android:layout_rowSpan="2/..." 设置子组件跨几行

二、UI组件

1、文本框组件TextView
    android:layout_width
    android:layout_height
    android:text     设置文本框的文字内容
    android:textSize="27sp/..." 设置文字大小
    android:textColor="#FF0000/..." 设置文字颜色        #+8位十六进制是ARGB，前两位代表透明度
    android:singleLine="true/..."    设置成单行文本框
    android:background

2、编辑框组件EditView
    android:hint="提示信息"    设置提示信息，输入框有输入则消失
    android:inputType="textPassword/number/..." 设置输入类型
    android:drawableLeft="图片资源"    在输入框左边绘制图像 同android:drawableStart
    android:drawablePadding 设置绘制的图像的内边距
    android:lines="5/..."    设置编辑框占的行数
    EditView.getText()可获得输入信息

3、按钮组件Button
    Button是TextView子类，可以使用其属性
    设置按钮单击事件监听器
        1、使用匿名类
        Button btn = (Button) findViewById(id);
        btn.setOnClickListener(new View.OnClickListener{
            @Override
            public void onClick(View v){...}
        });
        2、使用onClick属性
            在Activity编写一个包含View类型参数的方法
            在android:onClick属性指定上面定义的方法名

4、图片按钮ImageButton
    android:src="图片资源" 设置图片,ImageButton没有android:text属性
    android:background="#0000/..."    设置背景颜色，#0000表示完全透明

5、单选按钮RadioButton
    使用RadioGroup包裹RadioButton成为单选按钮组，可以实现单选的效果
    android:checked="true" 设置为默认选择

6、复选框CheackBox

7、日期选择器DatePicker

8、时间选择器TimePicker

9、计时器Chronometer
    android:format="已用时间：%s/..." 设置显示格式
    设置计时器功能
        setBase()    设置计时器起始时间
        setFormat()    设置时间格式
        start()        开始计时
        stop()        停止计时
        setOnChronometerTickListener()    为计时器绑定监听事件

三、UI高级组件

1、进度条ProgressBar
    style="?android:attr/progressBarStyleHorizontal..." 设置进度条样式
    android:max="100/..."    设置进度最大值
    android:progress="50/..."    设置当前进度

2、拖动条SeekBar
    SeekBar是ProgressBar的子类，可以使用其属性
    android:thumb="图片资源" 可以将拖动条的按钮设置为对应图片
    SeekBar.setOnSeekBarChangeListener(new SeekBar.OnSeekBarChangeListener(){...})设置拖动事件

3、星级评分条RatingBar
    android:numStars="10/..." 设置星星数量
    android:rating="5/..." 设置默认选择的星星数量
    android:stepSize="1/0.5" 设置一次选择的数量
    android:isIndicator="true/..." 设置是否不可改变
    RatingBar.getRating()获取用户选择的星数

4、图像视图ImageView
    android:src="图片资源" 设置显示的图片
    android:scaleType="fixXY/..." 设置XY独立缩放占满整个View
    android:adjustViewBounds="true/..." 设置是否根据边界自动缩放图片
    android:maxWidth/maxHeight="90dp/..."    只有设置了adjustViewBounds属性最大宽/长才有效果
    android:tint="#aaFF0000/..." 设置图像覆盖层

5、图像切换器ImageSwitcher
    ImageSwitcher is = (ImageSwitcher)findViewById(id);
    is.setOutAnimation(AnimationUtils.loadAnimation(MainActivity.this,android.R.anim.fade_out)); 设置淡出效果/可以根据文件设置效果
    is.setInAnimation(AnimationUtils.loadAnimation(MainActivity.this,android.R.anim.fade_int)); 设置淡入效果/可以根据文件设置效果
    is.setFactory(new ViewSwitcher.ViewFactory(){        设置视图工厂，可以设置监听器进行图像切换
        @Override
        public View makeView(){    
            ...
            return ImageView;
    }});

6、网格视图GridView
    android:numColumns="3/auto_fit/..." 设置列数
    android:gravity 元素位置
    android:verticalSpacing="5dp" 垂直间距
    显示图片需要适配器Adapter（包括ArrayAdapter、SimpleAdapter、SimpleCursorAdapter、BaseAdapter）实现
        1、SimpleAdapter
            布局文件cell.xml
                <LinearLayout>
                    <ImageView></ImageView>
                </LinearLayout>
            java代码
                private int[] pictures = new int[]{...};    传入图像id
                GridView gv = (GridView)FindViewById(id);
                List<Map<String,Object>> listitem = new ArrayList<Map<String,Object>>();
                for(int i=0;i<pictures.length;i++){
                    Map<String,Object> map = new HashMap<String,Object>();
                    map.put("image",pictures[i]);
                    listitem.add(map);
                }
                SimpleAdapter sa= new Simpleadpter(this,listitem,R.layout.cell,new String[]{"image"},new int[]{R.id.image});
                gv.setAdapter(sa);

        2、BaseAdapter
            private int[] pictures = new int[]{...};
            public class ImageAdapter extends BaseAdapter{        自定义适配器
                private Context context;                         上下文对象
                public ImageAdapter(Context c){    context = c;}
                @Override
                public int getCount(){return pictures.length;}
                @Override
                public Object getItem(int position){return null;}
                @Override
                public long getItemId(int position){return 0;}
                @Override
                public View getView(int position,View convertView,ViewGroup parent){
                    IamgeView iv;
                    if(convertView == null){iv = new ImageView(context);}
                    else{iv = (ImageView)convertView;}
                    return iv;
                }}

7、下拉列表框Spinner
    设置下拉选项
    1、android:entries="@array/n1" 设置下拉选项，要在数组资源中定义并引用
        创建array.xml资源文件        
        <resources>
            <string-array name="n1">
                <item>1</item>    第一个item是直接显示的选项
                <item>2</item>
                <item>3</item>
            </string-array>
        </resources>            

    2、通过适配器实现
        String[] ctype = new String[]{"1","2","3"};
        ArrayAdapter<String> adapter = new ArrayAdapter<String>(this,android.R.layout.simple_spinner_item,ctype);
        adapter.setDropDownViewResourse(android.R.layout.simple_spinner_dropdown_item);
        Spinner sp = findViewById(id);
        sp.setAdapter(adapter);
        String item = sp.getSelectedItem();  获得选择的列表项

8、列表视图ListView
    设置列表项
    1、同下拉框选项，设置数组资源文件并引用
        android:entries="@array/ctype"

    2、通过适配器设置列表项
        ListView lv = findViewById(id);
        lv.setAdapter(adapter);
        lv.setOnClickListener()设置点击列表项事件监听
        
9、滚动视图ScrollView
    垂直滚动视图<ScrollView></ScrollView>
    水平滚动视图<HorizontalScrollView></HorizontalScrollView>
    在滚动视图中只能有一个组件，想要放置多个组件需要用布局管理器将它们裹起来

10、选项卡
    使用选项卡有四个步骤
        1、在布局文件中添加TabHost、TabWidget、TabContent组件    
            TabHost和TabWidget有标记，TabContent通常使用FrameLayout
        2、编写各标签页的xml文件
        3、获取并初始化TabHost组件
        4、为TabHost对象添加标签页

    activity_main.xml中：
        <TabHost android:id="@android:id/tabhost">
            <LinearLayout>
                <TabWidget android:id="@android:id/tabs"></TabWidget>
                <FrameLayout android:id="@android:id/tabcontnet"></FrameLayout>
            </LinearLayout>
        </TabHost>
    tab1.xml和tab2.xml是两个页面的资源文件
    MainActivity.java中
        TabHost th = findViewById(id);
        th.setup()      初始化TabHost组件
        LayoutInflater inflater = LayoutInflater.from(this);    
        inflater.inflate(R.layout.tab1,th.getTabContentView());        加载xml布局文件，参数一是资源文件id，第二个是需要附加资源文件的根控件
        inflater.inflate(R.layout.tab2,th.getTabContentView());
        th.addTab(th,newTabSpec("tab1").setIndicator("选项卡1").setContent(R.id.left));
        th.addTab(th,newTabSpec("tab2").setIndicator("选项卡2").setContent(R.id.right));
