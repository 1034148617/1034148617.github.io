css伪元素

发表于 2019-11-23 更新于 2021-04-16 分类于 前端学习

在学习float浮动模型的时候，由于浮动流的原因，会导致漂浮的元素后面的元素排列会出现与预期不符的情况，需要添加上属性clear:both去清除浮动流来正常显示。可以选择在最后一个浮动元素后添加一个元素加上这个属性，但是会破坏原有结构。于是，可以在伪元素内加上该属性进行浮动流的清除。

每个标签在生成的时候，均带有两个伪元素，可以通过伪元素选择器将它们选择出来。
如：
    .classname::before{
        content:"";
    }
    .classname::after{
        content:"";
    }

伪元素的使用语法就是在选择器后加上::before或者::after就能找到对应标签的伪元素了，伪元素具有一个必须的元素content，即伪元素的文本，可以为空，但不能缺省。

使用伪元素进行浮动流的清除
其中wrapper为父级元素样式，content为子级元素样式
    .content{
        float: left;
    }
    .wrapper::after{
        content:"":
        display: inline-block;
        clear: both;
    }

这样就可以清除浮动流不对后面的元素样式造成影响，伪元素是标签自带的，也没有修改原有的结构。
