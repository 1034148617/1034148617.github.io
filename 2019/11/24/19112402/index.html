javascript中变量类型整理

发表于 2019-11-24 更新于 2021-04-16 分类于 前端学习

不可改变的原始值（栈数据）

包括Number，String,Boolean,undefined,null

引用值（堆数据）

包括array,object,function等

关于typeof的使用

1、使用方法 typeof(obj) 或者 typeof obj
2、作用：返回数据的类型，返回值的类型是字符串
3、返回值包括：number,string,boolean,object,undefined,function
（1）如果求的变量未定义，则typeof会返回”undefined”而不是报错
（2）输入的值为null时候，会返回object

类型转换问题

1、显示类型转换
Number() 返回数字类型，如果遇到不可以转成数字的就返回NaN
parseInt() 转换成整形，具有阶段功能 如parseInte(“100px”) —-> 100
parseFloat()
toString()
String()
Boolean()
2、隐式类型转换
（1）各类运算符，如+/- 以及== 和 ！=等
（2）isNaN() 这个函数本质是调用Number()判断返回值是不是NaN
3、不发生类型转换
（1）=== 和 ！==
（2）注意，NaN == NaN的结果是false
