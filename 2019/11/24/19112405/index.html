javascript包装类

发表于 2019-11-24 更新于 2021-04-16 分类于 前端学习

包装类的定义和作用

javascript中原始值自己是没有属性的，比如定义一个数字或者字符串，无法进行像对象Object一样给属性赋值的操作。但是却能调用一些方法，比如字符串 var str = “abc”,abc.length —>3,这个返回字符串长度的过程就使用了包装类。

    new String("abc").len = 3;delete this;

在调用length属性过程中，隐式地发生了上面的步骤：先是创建一个新的字符串对象，将其长度赋值为3，输出后就删除这个创建的对象，这就是为什么原始值也能使用属性的原因，js内部实现了包装类的规则。

包装类定义的函数

包装类定义的函数有三个：
    1、var num = new Number(123);
    2、var str = new String("abc");
    3、var bool = new Boolean("true");

这样显示得创建的是对应类型的对象，当访问原始值时便会隐式调用，进行包装类的过程。
