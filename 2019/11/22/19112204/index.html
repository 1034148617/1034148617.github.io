css简单常用选择器

发表于 2019-11-22 更新于 2021-04-16 分类于 前端学习

1、直接选择器

    .classname{}
    div{}
    #id{}
    *{}

包含几种基础的选择器：直接选择器、标签选择器、id选择器、通配符选择器

2、层级选择器

    .classname1 .classname2{}
    .classname1 > .classname2{}

也叫父子选择器，选择出具有对应父子关系的标签，第一行形式两个标签不一定是父子关系，只要1嵌套2就可以得到，第二行的选择器需要1和2是直接父子关系才可以筛选到

3、属性选择器

    [statuename]{}

筛选出具有此属性的标签，可以给属性赋值，进行更精确的筛选
