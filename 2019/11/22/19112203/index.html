css常用属性整理

发表于 2019-11-22 更新于 2021-04-16 分类于 前端学习

1、border

    border:4px solid red;
    borfer-left: 1px solid green;
    border-redius: 50%;

标签的盒模型属性之一，其中三个属性对应border-width、border-style、border-color，其中border-style常用的有solid（实线）、dotted（点状实线）、dashed（虚线）。还可以分别设置三个属性以及四边，如border-left/top/right/bottom。border-redius这是个特殊属性，用于设置边框的圆角值，50%显示为圆形边框。

2、background-color

    background-color: black;

设置背景颜色，值可使用rgb表示或者使用属性值

3、display

    display: block/inline/inline-block/none;

设置框模型，对应的效果为块/内联/内联块/消失

4、字体设置相关属性

（1）color

设置字体颜色，属性值同background-color

（2）font-size/weight

设置字体大小或者粗细

（3）font-style

设置字体样式

（4）font-family

设置字体集

（5）text-align
    text-align:left/center/right

设置文字水平方向，默认为left

（6）line-height
    line-height: 48px;

设置文字高度，若文字与其父标签等高，可实现垂直居中的效果

（7）text-decoration
    text-decoration:none/underline/line-through

文字带有线的效果，分别对应无/下划线/删除线

5、height和width

设置标签的宽和高，注意只有块级标签和内联块标签才能成功设置这两个属性，而内联标签inline不行

6、margin

    margin: 0 auto;
    margin-left: 0;
    margin-top: 10px;

margin是盒模型属性之一，代表含义是外边距，即边框和相邻元素的距离，可分别设定top/right/bottom/left，其默认设置方向为从top开始顺时针设置

7、padding

    padding: 50px;
    padding-left: -100px;

padding是盒模型属性值一，代表标签的内边距，即宽高和边框之间的距离.

8、opacity

    opacity: 0.5;

透明度设置，其值为0~1范围内的数

9、position

    position: fixed/absolute/relative

层模型属性

10、float

    float: right/left;
    clear: both;

浮动模型属性，生成浮动流，可用clear方法清楚浮动流

11、z-index

    z-index:5

层模型属性之一，设定标签的层数，默认为0，值大的在上层

12、list-style

列表样式，在使用ul-li的父子关系时，不想要列表样式可以通过此属性取消
