约束

发表于 2021-03-24 更新于 2021-04-16 分类于 数据库学习

1.主键约束（primary key）

①特性：非空、不可重复

②简写
create table stu(
sno char(4) primary key,
sname varchar2(100)
)
③具体
create table mark(
sid int,
cid int,
score int,
constraint pk_sidCid primary key（sid,cid）
)
④添加和移除主键
alter table stu add constraint pk_sid primary key(sid);
alter table stu drop constraint pk_sid;

2.唯一约束（unique）

①简写（建议使用）
create table stu(
sfz char(20) unique
)
②完整
create table stu(
sfz char(20),
constraint uk_sfz unique(sfz)
)

3.非空约束(not null)

使用方式
create table stu(
sfz char(20) unique not null
)

4.外键

①特性：外检矢量指向另一张表的主键或者唯一元素
②使用
create table stu(sid char(8) primary key,sname varchar2(100));
create table cou(cid char(8) primary key,cname varchar2(100));
create table mk(
sid char(8),cid char(6),score number,
constraint pk_sidCid primary key(sid,cid),
constraint fk_sid foreign key(sid) references stu(sid),
constraint fk_cid foreign key(cid) references cou(cid)
)

5.check约束（=where表达式）

使用
statue value check(…)
constraint ck_scr check(…)
