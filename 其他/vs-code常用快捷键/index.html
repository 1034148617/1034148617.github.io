二、常用快捷键

序号 快捷键 作用
1 Ctrl + X 剪切当前行
2 Ctrl + C 复制当前行
3 Alt + ↑/↓ 移动当前行向上/向下
4 Shift + Alt + ↑/↓ 向上/向下复制当前行
5 Ctrl + Shift + K 删除当前行
6 Ctrl + Enter 插入空行并移动光标
7 Ctrl + Shift + Enter 向上插入空行并移动光标
8 Ctrl + ]/[ 缩进/取消缩进一次
9 Ctrl + End 移动到文件最后
10 Ctrl + Home 移动到文件开头
11 Ctrl + / 注释/取消注释
12 Shift + Alt + A 块注释/取消注释
