2.4 其他
  • with(online = on|off ) 创建索引时用户可以访问表中的数据
  • with(pad_index = on|off fillfactor = 80) 设置叶级别页的填充百分比
  • with(allow_row_locks = on|off | allow_page_locks = on |off) 是否允许行锁/页锁
  • with(data_compression = row | page ) 可用于压缩索引大小
