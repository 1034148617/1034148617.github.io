sql查询语句

发表于 2021-03-19 更新于 2021-04-16 分类于 数据库学习

查询select执行顺序

select------5
from--------1
where-------2
group by----3
having------4
order by----6

from子句

①将瓷盘中的表格通过i/o读入计算机内存，并让中的数据在内存中生存一个实例
②将多个实例进行笛卡尔相乘
③执行for循环，每次从内存实例中取出一条记录，交给后续子句处理

select子句

①select sname 姓名 from student;        //取别名
②select distinct sage from student;  //去重
 select distinct sage,ssex from student; //去掉重复的组合

where子句

①支持比大小 = != <> > < >= <=
    select sname from student where sage = 19;
②支持逻辑运算 not and or(优先级降序排列)
    select * from student where snativeplace='福建'and ssex='女';
③支持连续的范围表达式between(效率高)
    select * from student where sage between 19 and 24;
④支持非连续的表达式in(效率低)
    select * from student where sage in (19,21,24);
⑤支持模糊匹配
    在like的作用下, _表示一个任意字母
    在like的作用下, %表示一个任意多个字母
⑥支持对空(null)的匹配
    select * from student where sage is null;
    select * from student where sage is not null;
    select * from student where not sage is null;

group by子句（统计）

①归类
    select ssex,max(sage),min(sage),avg(sage),sum(sage),count(*)
    from student group by ssex;
②等价于for循环

having子句（对分组判断）

select sid from mark group by sid habing min(cmark>60);
select sclass,count(*) from student group by sclass,ssex having     ssex='男；

order by子句（排序）

select * from student order by sage asc; //默认升序，asc可省略
select * from student order by sage desc; //降序
select / from student order by sage desc,sid;

空值是最大值,不想要空值可以用where .. is not null筛选
